# DreamentrixAPI
API for dreamentrix 

Installation
  1. cd to project_dir
  2. python -m virtualenv venv
  3. type this command in terminal:
     >>> . /venv/bin/activate <<<

     now you mush see (venv) in front ur username in terminal like this >> (venv)username &: _ <<

  4. install package
     >> pip install -r requirements.txt

  5. start api
     in root dir 
     >> python app.py
